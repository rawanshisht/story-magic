**Storybook Generator** is a magical web application that uses AI to create personalized, illustrated stories for children. By combining the power of OpenAI's text and image models with a kid-friendly interface, parents can generate unique adventures starring their own children.

## ‚ú® Features

-   **Personalized Protagonists**: Create stories featuring your child by specifying their name, age, appearance, and interests.
-   **Moral-Driven Storytelling**: Choose from a variety of moral lessons to guide the narrative.
-   **Consistent AI Illustrations**: Uses advanced prompting techniques to maintain character consistency across watercolor-style illustrations.
-   **Kid-Friendly Design**: A playful, safe, and accessible UI designed specifically for children aged 5-10.
-   **PDF Export**: Download the generated stories as beautifully formatted PDF books.
-   **Story History**: Save and revisit generated stories in your library.
